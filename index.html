Guia de Git e GitHub

Aprenda controlo de versão e colaboração de forma simples

O que é Git?

Git é um sistema de controlo de versão que permite guardar o histórico do código, voltar atrás em alterações e trabalhar em equipa.

Instalação do Git

  • Windows: Baixar em https://git-scm.com
  • Linux: sudo apt install git
  • Mac: brew install git

Configuração Inicial

git config --global user.name "Teu Nome"
git config --global user.email "teu@email.com"

Criar Projeto

mkdir meu-projeto
cd meu-projeto
git init

Comandos Básicos

git add .
git commit -m "mensagem"
git status
git log

Ligar ao GitHub

git remote add origin https://github.com/user/repositorio.git
git push -u origin main

Fluxo de Trabalho

  • Editar código
  • git add .
  • git commit
  • git push

Dicas Importantes

  • Fazer commits frequentes
  • Usar mensagens claras
  • Testar antes de fazer push
