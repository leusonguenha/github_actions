Quem somos?

Somos o Grupo II, uma equipe dedicada ao aprendizado e aplicação de práticas DevOps. Este projeto demonstra como usar GitHub Actions para automatizar o deploy de um site simples.

Nossa missão

Promover a automação no desenvolvimento de software, reduzindo erros manuais e melhorando a eficiência.
